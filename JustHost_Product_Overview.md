# JustHost (ARTURO) — Product & Project Overview

## 1. What is JustHost?

JustHost is an AI-powered operating system for independent hotels.

At its core is **ARTURO**, a multi-agent orchestration system that coordinates 19 specialized assistants across guest communication, operations, and administration.

It is designed to help small-to-mid-sized hotels run efficiently without needing large teams.

---

## 2. Core Idea

Independent hotels face operational overload:
- Too many manual tasks
- Small teams handling everything
- No scalability without hiring more staff

JustHost solves this by:
> Automating repetitive work while preserving human hospitality.

---

## 3. Key Value Proposition

**“Your hotel runs. Even when you don’t.”**

### What this means:
- Guests get instant responses (24/7)
- Operations stay synchronized automatically
- Admin work is reduced drastically
- Owners regain time and control

---

## 4. The Product: ARTURO

ARTURO is not a chatbot.

It is an **orchestrator** that:
- Understands context
- Delegates tasks to specialized agents
- Syncs data across systems
- Escalates only when needed

---

## 5. The 3 Core Modules (MVP)

### 1. Guest Experience

### 2. Operations Excellence

### 3. Finance & Admin

---

## 6. How It Works

### Step 1 — Install
Connect existing tools:
- PMS
- Communication channels
- Accounting software

No data migration required.

### Step 2 — Train
- Define tone and policies
- Input business rules
- ARTURO learns from real interactions

### Step 3 — Automate
- Tasks handled automatically
- Dashboard for monitoring
- Human intervention only when needed

---

## 7. Decision System

ARTURO operates on 3 levels:

### Automatic (80%)
Routine tasks handled instantly.

### Suggested (15%)
Requires quick approval.

### Escalated (5%)
Complex or sensitive cases handled by humans.
